Softmax classifier

This was a group project I did for COMP 551: Applied Machine Learning at McGill University. I worked in a group setting to implement Softmax regression and then apply it to a wide variety of datasets for classification. My contribution was two-fold. Firstly I implemented the Softmax function itself which provided an interesting mathematical challenge. Secondly, I modified our gradient descent optimizer to use Adagrad optimization as described in this paper

Designing an implementation of Softmax turned out to be an interesting challenge mathematically becase the Softmax function is a mapping from R^n to R^n and therefore its derivative is represented by a n by n Jacobian matrix with n*n entries, but in order to be compatible with gradient descent, I needed a 1 by n vector, and ultimately was able to determine that the Jacobian diagonal was what I needed, however this with rather non-trivial.

Implementing Adagrad was much more straightforward and the only problem I had a chance to solve was working with the software engineering decisions my other teammates had made and working with those. Having to do this not only taught me about Adagrad but about communication in a software engineering context as well.
