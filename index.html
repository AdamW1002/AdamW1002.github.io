COVID analytics

This was a group project I did for COMP 551: Applied Machine Learning at McGill University. We took data about search trends for items related to COVID-19 that were organized by time and region and analyzed them. In this project my contribution was to implement and run the machine learning experiments, and then perform validation by either testing the model on data from a different time or region. I used the scikit-learn library's implementation of KNN and decision trees in order to gain experience with industry level software.
